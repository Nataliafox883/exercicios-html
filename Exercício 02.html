Imagens

Nessa parte do Exercicio, aprendemos baixar o GIMP,
editar as imagens pra por no site e além disso,
escolher imagens com licensa para usar,editar e comercializar.
Sendo elas com fundo colorido e transparente.

- Colocar imagens a partir de links

Para inserir as imagens a partir de links, escolhe-se a imagem
com toda a licensa e copia-se o link e cola.
O recomendado é que se escolha uma imagem já pequena.
